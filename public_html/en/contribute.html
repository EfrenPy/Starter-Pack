Contribute to CERN Starter Pack

This site is maintained by CERN community members. Your feedback and contributions make it better for the next newcomer.

How You Can Help

  • Report outdated information: If you find any content that is no longer accurate, let us know so we can update it.
  • Suggest new resources: Know an official CERN link or external resource that would help newcomers? We'd love to add it.
  • Share your experience: Your onboarding story can help identify gaps in our guides and improve them for future newcomers.
  • Fix errors: Spotted a typo, broken link, or incorrect detail? Report it or submit a fix directly.

Get in Touch

  • GitHub: Open an issue or submit a pull request on our GitHub repository. This is the preferred way to contribute.
  • Email: If you prefer, you can reach us by email. Contact details are available on the GitHub repository page.

Contribution Guidelines

  • Official sources only: Please suggest or reference only official, verified sources (CERN, government, institutional).
  • Keep it factual: Avoid personal opinions on employers, landlords, or service providers. Stick to objective, verifiable information.
  • Be helpful: Focus on information that is practical and actionable for someone arriving at CERN for the first time.
